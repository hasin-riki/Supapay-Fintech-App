# Supapay-Fintech-App
An application that provides basic financial services, such as mobile banking and customer support.

## Features
- Signup
- Hard and Soft Login
- Forgot password
- OTP Authentication
- Transactions simulation
- Customer support chat
- Credit card
- ATMs locations
- QR scanner
- Clipboard
- Light/dark themes

## User Roles
- User
- Support

## Technology
- App: Flutter 
- Database: Firebase
